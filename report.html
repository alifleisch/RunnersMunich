Report on Runners Munich Website

Introduction:

The Runners Munich website is designed to serve as a digital hub for a community-driven sports organization focused on running, climbing, and strength training. The website structure is user-friendly, providing easy navigation across five main pages: Home & About Us, Running, Climbing & Strength, Join Us, and Contact & Location. Each page is dedicated to different aspects of the organization, from detailed program descriptions to membership information and contact details.

Inspiration:

  • Adidas Runners Munich(Adidas Runners Munich): The Adidas Runners community in Munich provided significant inspiration with its focus on community building, inclusivity, and the holistic approach to running.
  • Boulderwelt München Ost(Boulderwelt München Ost): The climbing community at Boulderwelt München Ost inspired the integration of climbing and strength training into our programs.

Accessibility:

  • Contrast, Readability, and Dark Theme: The site enhances accessibility through high-contrast text, making the site easier to use for people with vision problems and those who find bright screens uncomfortable.
  • Alt Text for Images: Descriptive alt text helps people who can't see images understand what they're about.
  • Keyboard Navigation: The website is navigable via keyboard, ensuring accessibility for users with mobility impairments.

Usability:

  • Responsive Design: The website is fully responsive, providing an optimal viewing experience across various screen sizes.
  • Clear Navigation: A consistent and straightforward navigation menu across all pages allows for easy movement between sections.
  • Engaging Content: The use of engaging visuals and informative text ensures that users can easily find and understand the information they are looking for.

Learning:

  • Responsive Web Design: Implementing responsive design principles was important to ensure the site's functionality on different screen sizes.
  • JavaScript Interactivity: Learning to create interactive elements like the theme toggle button and the image carousel enhanced the site's dynamic features.
  • Performance Optimization: Learning about and applying techniques for optimizing website performance, including faster load of the video was essential for enhancing user satisfaction.

Evaluation I:

The integration of community stories and testimonials was particularly successful, as it humanized the organization and built a connection with the audience. Additionally, the implementation of the newsletter form validation ensured a better user experience by preventing form submission errors. The dark theme toggle feature enhanced the site's accessibility and user preference options, making the website more inclusive and comfortable for night-time browsing. The dynamic switching of testimonials every 5 seconds added an engaging and lively element to the site, keeping the content fresh and interesting. Moreover, the use of JavaScript to append the footer to each HTML page automatically was a big achievement, as it improved the development process and ensured consistency across the site without the need to manually copy and paste the code into every page.

Evaluation II:

The site could benefit from more modern styles, animations, and CSS features could enhance the visual appeal and user engagement. For instance, smooth transitions between pages and more interactive elements could create a more pleasant experience. Implementing a more varied color palette and typography could also elevate the site's aesthetic and align it with current design trends. It's important to note that achieving a sophisticated style requires a good sense of design and aesthetics, which often comes with experience. As I am at the beginning of my learning path, I recognize the need for further development in this area. In the future, I look forward to improving my skills and applying more advanced design principles to create even more engaging and visually appealing websites.

Resources:

To build this website, I used many tools and resources to make sure it works well, looks good, and is easy to use. Here's a list of sources of information, libraries, plugins, code snippets, and tools:

  • HTML5 and CSS3: The core structure and styling of the website were built using HTML5 and CSS3, adhering to the latest web standards for responsiveness and design.
  • JavaScript: For dynamic content and interactivity, such as the theme toggle button, image carousel, and testimonial cycling, JavaScript was extensively used.
  • W3Schools(https://www.w3schools.com): This educational website provided tutorials and code examples for HTML, CSS, and JavaScript, which were instrumental in implementing various features like the navigation bar and responsive design.
  • Font Awesome(https://fontawesome.com): This icon library was used to enhance the user interface with scalable vector icons that could be customized with CSS.
  • Google Fonts(https://fonts.google.com): The website utilized fonts from Google Fonts for a consistent and accessible text presentation across different devices and browsers.
  • jQuery(https://jquery.com): Although not heavily relied upon, jQuery was used for certain DOM manipulations and to simplify event handling, particularly in the contact form validation.
  • Responsive Web Design Tester (Chrome Extension): This tool helped in testing the website's responsiveness across various device sizes to ensure a seamless user experience.
  • Color Inspiration from Boulderwelt München Ost(https://www.boulderwelt-muenchen-ost.de): The color schemes and patterns for the website's design were inspired by the vibrant and dynamic aesthetic of the Boulderwelt München Ost website, contributing to the site's visual appeal and thematic consistency.
